Routes

  • /bar, /pie, /line - graph variants

Query parameters

  • values, v - (required) list of integer values to graph. /pie expects only positive values.
  • size, s - (optional) the size of the resulting image (max 1024)
  • color/colors, c - (optional) color(s) to use when graphing in hex notation (e.g. ff0000). /line expects color. /bar and /pie expect colors. Colors list will be rotated when less than values count of colors are defined
  • donut - (optional) Only affects /pie. If set to true, pie chart is drawn as a donut.

Examples

Simple bar graph with randomly defined color: http://{{ .host }}/bar?size=100&values=1,2,3
Simple bar graph with randomly defined color using short notation: http://{{ .host }}/bar?s=100&v=1,2,3
Simple bar graph with red color: http://{{ .host }}/bar?s=100&v=1,2,3&colors=ff0000
Simple bar graph with different defined colors: http://{{ .host }}/bar?s=100&v=1,2,3&c=ff0000,00ff00,0000ff
Simple bar graph with different defined colors (colors will cycle): http://{{ .host }}/bar?s=100&v=1,2,3,5,8,13&c=ff0000,00ff00,0000ff
Line graph: http://{{ .host }}/line?s=100&v=1,2,3,5,8,13&c=ff0000
Pie graph: http://{{ .host }}/pie?s=100&v=1,2,3,5,8,13
Pie graph (donut): http://{{ .host }}/pie?s=100&v=1,2,3,5,8,1&donut=true
